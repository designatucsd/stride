# Stride Career Fair Website
## Set Up
1. Install Jekyll.
```
$ gem install jekyll
```
2. Install project dependencies.
```
$ bundle install
```
## Development
Serve the site with:
```
$ bundle exec jekyll serve
```
View the site at http://localhost:4000.
